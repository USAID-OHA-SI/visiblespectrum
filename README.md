# visiblespectrum
A package to scrape the UNAIDS NAOMI Spectrum HIV sub-national estimates viewer.

Package is currently in development.

```{r}
pull_naomi(
  countries = "all",
  indicators = "all",
  age_groups = "standard",
  sex_options = "all",
  periods = "recent",
  max_level = "none",
  verbose = FALSE
)
```
